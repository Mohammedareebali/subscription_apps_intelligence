# Raw data — place Kaggle CSVs here

This folder is gitignored. Drop two files here before running any notebook:

1. **`googleplaystore.csv`** — from [Kaggle: Google Play Store Apps](https://www.kaggle.com/datasets/lava18/google-play-store-apps) (`Lavanya Gupta`, ~10k rows)
2. **`googleplaystore_user_reviews.csv`** — same Kaggle page, companion file (~64k rows)

Both files are licensed CC0 on Kaggle.

```
data/raw/
├── googleplaystore.csv
└── googleplaystore_user_reviews.csv
```

The dataset is a **2018 snapshot**. This is disclosed in the top-level README and `reports/methodology.md`.
